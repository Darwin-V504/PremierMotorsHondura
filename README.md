Proyecto de Vinculación Comercio Electrónico con empresa Premier Motors Honduras

Para inicializar proyecto de manera local:
-Usar VS code preferiblemente
-Tener instalado node.js 
-Extraer todo del zip
-Abrir carpeta en VS code
-Utilizar en terminal npm install para instalar librerías-dependencias
-inicializar en terminal npx expo start para lanzar en aplicación expo App (Android)
